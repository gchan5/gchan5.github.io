Game of Life

This is a parellel implementation of the Game of Life, a cellular automaton devised by John Conway in 1970. This game is "played" on a board, or a world, consisting of m rows, and n columns. An initial generation (generation 0) is either entered by the user, or randomly generated. This was created with C and pthreads, in USF's Introduction to Parallel Computing course.

  • Language(s): C
  • Date: Fall 2015
  • Link: Github

Card Suite

Play against mischievous friends or brilliant bots, or even a combination of both with a customizable bot and player selection system.

Song Finder

This academic project was created with the intent to provide a service where users can search for songs similar to a given tag, artist, or song title. Song Finder is uses Java, Jetty, JDBC, MySQL, HTML, and CSS. Amazon Web Services was used to deploy this web application. Information stored in Song Finder's relational databse comes from the Last.fm data subset. Song Finder allows users to log in, track their searches, favorite songs, search for similar songs, and more. Artist information is obtained through the use of Last.fm's REST API.

  • Language(s): Java, HTML, CSS
  • Date: Spring 2016
  • Link: Song Finder
